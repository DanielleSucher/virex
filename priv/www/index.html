Vim Regex Quick Reference (in default/magic mode)

  • [abc] A single character: a, b, or c
  • [^abc] Any character except: a, b, or c
  • [a-zA-Z] Any single letter
  • [:alnum:] Any single letter or digit
  • [:punct:] Any single punctuation character
  • \_^ Start of line
  • \_$ End of line
  • . Any single character
  • \. Literal dot
  • \s whitespace character
  • \S non-whitespace character
  • \d digit
  • \D non-digit
  • \w word character
  • \W non-word character
  • \< left word boundary
  • \> right word boundary
  • \(...\) Capture everything enclosed
  • a\|b a or b
  • a\? Zero or one of a
  • a* Zero or more of a
  • a\+ One or more of a
  • \{3} Exactly 3 of a
  • \{3,} 3 or more of a
  • \{3,6} Between 3 and 6 of a
  • r\%[ead] Longest available of: r, re, rea, read
  • \(.\)a\1 'yay', 'dad', 'xax', &c
For detailed documentation on vim's regex, check out vimregex.com, the vim docs, and the vim tips wiki.
